# File Lion uploader

Download video from eksternal URL and upload to File Lion via REST API

## Installation

Tested on : 
- python `>=3.8`

Install modules
```bash
pip install -r requirements.txt
```

## Run app
Command
```bash
python run.py --url="{VIDEO_URL}" --filename="{FILENAME.mp4}"
```
